What it does

Built for accessible,
real-time communication.

🤟
ASL Gesture Recognition
Detects and interprets American Sign Language gestures in real time using a trained deep learning model.
✋
Hand Tracking
MediaPipe and cvzone track hand landmarks frame-by-frame with high precision, even in low-light conditions.
📹
Live Video Integration
Integrates directly with Google Meet via screen capture, processing the video stream without interruption.
⚡
Real-Time Processing
Frame extraction and inference happen in milliseconds, keeping gesture-to-text translation smooth and responsive.
🎥
WebRTC Video Calling
Peer-to-peer video calling built on WebRTC for low-latency, browser-native communication between participants.
🧠
Trained ML Model
A TensorFlow/Keras model trained on ASL gesture datasets, optimised for real-world lighting and hand variability.

Getting started

Up and running
in four steps.

1
Clone the repository
Clone MuteVerse to your local machine.
git clone https://github.com/AMit090912/MuteVerse
cd muteverse
2
Create a virtual environment
Isolate dependencies with a Python virtual environment.
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
3
Install dependencies
Install all required Python packages.
pip install -r requirements.txt
4
Run the application
Start MuteVerse and open your browser.
python main.py

→ http://localhost:5000
